# Smart Hospital

**Progetto per il corso di Ingegneria dei Sistemi Distribuiti** *Università degli Studi di Catania - Corso di Laurea Magistrale in Informatica*

## L'idea alla base del progetto

L'idea alla base di questo progetto è sviluppare "Smart Hospital", un sistema distribuito pensato per gestire in modo semplice ed efficiente un ente ospedaliero. 

L'obiettivo principale è fornire a tutte le entità coinvolte (pazienti, personale sanitario e amministrativo) un'interfaccia comoda e dedicata per svolgere le proprie attività quotidiane. Nello specifico:

* **Personale Sanitario (Medici e Infermieri):** Il sistema offre allo staff una piattaforma per snellire il lavoro di tutti i giorni, dai ricoveri all'assegnazione dei turni, fino alla prenotazione degli esami clinici.
* **Pazienti:** È stato pensato un portale che permette ai degenti di consultare comodamente, in tempo reale e in totale privacy, la propria cartella clinica e i referti.
* **Amministratori:** Un pannello di controllo per gestire le anagrafiche degli utenti e monitorare le statistiche di occupazione dei reparti in tempo reale.
